# 🎭 TrueMafia - Браузерная Мафия

Полноценный браузерный порт VK-бота мафии с real-time геймплеем, ботами и экономикой.

## 📁 Структура проекта

```
mafia-site/
├── frontend/          # HTML/CSS/JS клиент
│   ├── index.html     # Лобби
│   ├── game.html      # Игровой стол
│   ├── profile.html   # Профиль игрока
│   ├── css/           # Стили и анимации
│   │   ├── style.css
│   │   └── animations.css
│   └── js/            # JavaScript логика
│       ├── websocket.js   # WebSocket клиент
│       ├── game.js        # Игровая логика
│       └── ui.js          # UI и анимации
│
├── backend/           # Node.js сервер
│   ├── server.js      # Express + Socket.io
│   ├── game-engine.js # Игровой движок
│   ├── bot-ai.js      # AI ботов
│   ├── db.js          # SQLite база данных
│   └── auth.js        # JWT авторизация
│
├── assets/            # Ресурсы
│   ├── roles/         # Изображения ролей
│   ├── sounds/        # Звуковые эффекты
│   └── animations/     # CSS/JS анимации
│
└── data/              # База данных SQLite
```

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd mafia-site
npm install
```

### 2. Настройка окружения

```bash
cp .env.example .env
# Отредактируй .env файл
```

### 3. Запуск сервера

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

### 4. Режим разработки (с авто-перезагрузкой)

```bash
npm run dev
```

## 🎮 Как играть

1. Открой `http://localhost:3000`
2. Введи имя и присоединись к лобби
3. Жди пока наберётся минимум 4 игрока (или нажми "Форс-старт")
4. Игра начинается автоматически!
5. Боты заполняют оставшиеся слоты до 12 игроков

## 🤖 Боты

Боты имеют:
- **Персональность** (агрессивный, тихий, дружелюбный, подозрительный, хитрый)
- **Память** о сообщениях, голосованиях, подозрениях
- **Логику** ночных действий и голосований
- **Чат** — пишут сообщения днём с анализом ситуации

## 🗄️ База данных

SQLite с таблицами:
- `users` — профили, баланс, статистика
- `settings` — настройки игры
- `cooldowns` — кулдауны работы/кражи/бонуса
- `bank` — банковские вклады (16% за 48ч)
- `shop_items` — купленные предметы
- `role_stats` — статистика по ролям
- `achievements` — достижения
- `game_history` — история игр

## 🔧 API Endpoints

### Auth
- `POST /api/register` — Регистрация
- `POST /api/login` — Вход

### Profile
- `GET /api/profile/:userId` — Профиль игрока
- `GET /api/top` — Топ-10 по балансу

## 🔌 WebSocket Events

### Client → Server
- `join_lobby` — Присоединиться к лобби
- `start_game` — Начать игру
- `night_action` — Ночное действие
- `vote` — Голосовать
- `skip_vote` — Пропустить голосование
- `defence_vote` — Вердикт суда
- `chat` — Отправить сообщение

### Server → Client
- `lobby_update` — Обновление лобби
- `player_joined/left` — Игрок присоединился/вышел
- `role_assigned` — Роль выдана
- `phase_change` — Смена фазы
- `night_action_request` — Запрос действия
- `vote_start/update` — Голосование
- `night_results` — Результаты ночи
- `game_over` — Конец игры
- `chat_message` — Сообщение в чате
